# UDAAN 2.0 — Breaking Bihar's Industrial Stagnation

A board-grade government strategy deck (1 title + 10 content slides) prepared for
Government of Bihar senior officials and the GILP Fellowship 2026 jury.

**UDAAN** = *Unleashing District-level Agro-manufacturing and Advanced iNdustrialisation.*

Governing thought: *Bihar's stagnation is an execution gap, not a policy gap.* The deck
diagnoses six structural bottlenecks, then wires proven models onto Bihar's own delivery
rails (JEEViKA, Makhana Board, FPOs, ITIs, the migrant diaspora) through a five-pillar,
24-month framework.

## Deck structure

1. Title
2. Context — the growth–development paradox + why now
3. Root-cause 1/2 — the system (hard-infrastructure gaps)
4. Root-cause 2/2 — institutional constraints + bottleneck→pillar bridge
5. Solution 1 — Land & plug-and-play readiness (P1)
6. Solution 2 — Women-led & returnee enterprise engine (P2)
7. Solution 3 — Distributed cluster activation / OBOP (P3)
8. Solution 4 — Skills-to-job pipeline (P4)
9. Solution 5 — Finance & governance acceleration (P5)
10. Execution roadmap + KPI dashboard
11. Risks, mitigations & references

## Design system

- **Fonts:** Bebas Neue (headers, ≤32 pt) + PT Sans (body).
- **Palette (3 brand colours only):** deep navy `#0F2A43`, teal `#0E8C84`, amber `#E0901F`,
  plus white / grey neutrals.
- **Spacing:** 0.55″ margins, 0.2–0.3″ inter-block gaps, soft navy card shadows.

## Rebuilding from source

Requires Node.js with `pptxgenjs`, `react-icons`, `react`, `react-dom`, `sharp`.

```bash
node gen_icons.js     # rasterises the icon set into assets/
node build.js         # writes UDAAN_2.0.pptx
```

Optional render to PDF/images via LibreOffice:

```bash
soffice --headless --convert-to pdf UDAAN_2.0.pptx
pdftoppm -jpeg -r 140 UDAAN_2.0.pdf slide
```
